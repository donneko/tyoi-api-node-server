# Static Tyoi Starter

Tyoi でHTML、CSS、JavaScriptを配信する静的サイト用テンプレートです。

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`: ブラウザを開いて開発サーバーを起動
- `npm run typecheck`: ブラウザJavaScriptを型チェック
- `npm run build`: 配布前チェック
- `npm start`: ブラウザを開かずにサーバーを起動

画面は `public/main/`、ポートなどは `tyoi.config.js` から変更できます。
