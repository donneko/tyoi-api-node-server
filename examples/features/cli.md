# CLI 最小コマンド

```bash
tyoi create my-app --template basic-ts # 新しいフォルダーに作成
tyoi init my-app --template basic-ts   # 現在のフォルダーに作成
tyoi config                            # 設定ファイルを追加
tyoi info                              # 適用される設定を表示
tyoi run                               # 設定ファイルから起動
tyoi run --port 3001 --open            # 起動オプションを上書き
tyoi help
tyoi --version
```
